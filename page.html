Tips para la Lectura

Algunos tips para poder mejorar



Lee: Leer constantemente es importantísimo. Se debe practicar para que
poco a poco se pueda mejorar la comprensión. Es recomendable leer
temas que te interesen, así la lectura no se vuelve aburrida y se pueda terminar el libro


Lenguaje: Es mejor comenzando a leer textos sencillos con un vocabulario
coloquial, luego leer texto más especializado. Se puede alternar la
complejidad del lenguaje. De esta manera se podra aumentar el
vocabulario y acostumbrarás a diferentes textos


Notas: Algunos libros son muy complejos, sea por la cantidad de personajes
o datos importantes que se necesita recordar para continuar leyendo. Es
recomendable que se haga algunas anotaciones, así se podrá repasar los
hechos que ya leídos


Leer despacio: Para mejorar la comprensión lectora, se debe leer despacio
cada línea. Así podrás entender oración por oración


Pausas: Es Bueno realizar una pausa cada párrafo, para continuar la
lectura. Se analiza lo leído para asegurar que lo entendiste


Tranquilidad: Es necesario que se lea en un lugar tranquilo para poder
concentrarse. Toda la atención debe estar en la lectura. Leer en silencio y
sentado, disfrutar lo que se lee así es más fácil comprenderla


Preguntas: Cuando cuando se termine de leer, se debe realizar preguntas
sobre el entorno del texto. La idea es que se recuerde lo sucedido. Así
podrás darte cuenta si la comprendiste


Diccionario: Es importante tener un diccionario cada vez que leamos.
Algunas veces no conoceremos el significado de algunas palabras, por esa
razón no podremos interpretar la lectura de manera correcta. El
diccionario ayudará a saber el significado y además podemos aprender
nuevas palabras
